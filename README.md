# Electrical Measurement Data Analysis Using MATLAB

## Overview
This project demonstrates basic electrical data analysis using MATLAB.
Voltage, current, and temperature signals were analyzed to understand
system behavior and derive power characteristics.

## Methodology
- Generated time-based electrical signals
- Visualized voltage, current, temperature, and power
- Calculated mean and peak electrical parameters
- Documented findings in a structured report

## Tools Used
- MATLAB

## Key Outcomes
- Electrical signal visualization
- Power calculation from voltage and current
- Basic system validation through parameter analysis
